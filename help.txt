Help git
Первоначально надо инициировать гит командой git init
После следует авторизовать если ещё не авторизованы. git config --global user.name <git nickname>
                                                     git config --global user.email <email@email.lu>
Команда git status показывает какие файлы были созданы или изменены в проекте
git add <name> добавляет или изменяет файл в проекте. Если после add поставить точку (.) то это добавит все файлы.
git commit -m "text commit" добавляет коммит. -m - message "текст сообщения".
git branch - отображает существующие ветки гит.
git branch <name> создаёт новую гит ветку для работы другому пользователю.
git checkout <name> перейти в ветку
git merge <name> производит слияние веток. При этом надо находится в главной ветке(master(or main))
git branch -D <name> удаляет гит ветку.
в файле gitignore надо написать название файла и расширение что бы добавить в гит игнор. Служит для того что бы 
не заливать нужные и/или тайны файлы что не для общего доступа.
git clone команда  клонирует чей-то гит(проект) для совместной работы, изменения или работы.
